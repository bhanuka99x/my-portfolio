## 🏗️ Portfolio Architecture

This repository contains the source code for my professional portfolio, which is itself a showcase of high-end web engineering.

### **Key Features**
- **Dynamic Backgrounds**: Custom-engineered `Galaxy`, `Beams`, and `LightPillar` effects.

### **Running Locally**
1. Navigate to the frontend directory: `cd frontend`
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
